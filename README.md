qpcrnlme
========

R package for quantitative real-time PCR analysis using sigmoidal mixed models

#### More information can be found in:     

Gerhard, D., Bremer, M. & Ritz, C., 2014. Estimating marginal properties of quantitative real-time PCR data using nonlinear mixed models. *Biometrics*, **70**(1), pp.247â€“254. Available at: http://onlinelibrary.wiley.com/doi/10.1111/biom.12124/abstract.

----------------------

The package can easily be installed, using the devtools package:
```
library(devtools)
install_github("daniel-gerhard/qpcrnlme")
```
